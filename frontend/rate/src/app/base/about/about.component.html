Witaj!

Serwis ten jest projektem szkoleniowym zrealizowanym w celu doskonalenia umiejętności tworzenia alpikacji internetowych z wykorzystaniem frameworka Angular 9.
Do wizualizacji danych użytko biblioteki D3.js 4.0.
Stylowanie wykonane z wykorzystaniem bibliteki Bulma oraz preprocesora LESS.
Strona posiada także prosty backend napisany w Django REST Framework w celu bezpiecznego przechowywania kluczy prywatnych oraz uniknięcia ograniczeń CORS.
Aplikacja wykorzystuje następujące API do pobierania aktualnych oraz historycznych danych:
  • AKCJE - kursy bieżące: IEX Cloud API
  • AKCJE - dane historyczne oraz PARY WALUTOWE - kursy bieżące oraz dane historyczne: Finnhub Stock API
  • INDEKSY - kursy bieżące oraz dane historyczne: CURRENCY.COM
  • ETFy - kursy bieżące oraz dane historyczne: TwelveData
  • KRYPTOWALUTY - dane historyczne: CoinGecko
  • WIADOMOŚCI: NewsAPI
    • Wszystkie wymienione źródła są darmowe lub posiadają darmowy dostęp dla niekomercyjnych użytkowników.
    Wiąże się to niestety z faktem, że wykorzystywane API nie są niezawodne. Posiadają one albo limity żądań na jednostkę czasu albo nie gwarantują płynności działania. Również aktualizacja danych w czasie rzeczywistym w przypadku niektórych serwerów jest mocno umowna (aplikacja pobiera aktualne ceny zawsze co 60s, jednak nie wszystkie źródła aktualizują je z taką częstotliwością). Na stronie mogą zatem pojawiać się sporadyczne problemy z wyświetlaniem niektórych elementów na skutek wykorzystania limitu danych (limity są minutowe oraz dobowe - te drugie ciężko przekroczyć) lub problemów z nawiązaniem połączenia.
